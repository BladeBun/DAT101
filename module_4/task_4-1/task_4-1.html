1. [Why is object-oriented development so important in programming?]

    2. [Objektorientert utvikling (OOP) er viktig i programmering fordi det gir en strukturert tilnærming til å organisere kode på en måte som speiler virkelige objekter og deres interaksjoner. OOP gjør det lettere å håndtere kompleksitet ved å dele opp programmet i små, uavhengige enheter (objekter).]

  2. [In Task 1, we create a static object called "AccountType." Explain what this object is used for and how this type of object can help us in software development.]

    3. [Et statisk objekt som "AccountType" kan brukes til å definere en felles struktur eller egenskaper som er relatert til forskjellige kontoer i et program. Ved å bruke statiske objekter kan vi definere verdier eller metoder som er felles for alle kontoer, og dermed unngå duplisering av kode.]
